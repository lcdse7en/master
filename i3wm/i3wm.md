### i3wm
#### 1.ubuntu install i3wm
- ubuntu系统默认安装的桌面环境是gnome
```bash
sudo apt-get install xfce4
sudo apt-get install i3
sudo apt-get install lightdm
```

#### 2.i3wm hotkey
<p>i3wm中默认Mod1为Alt，Mod4为windows键，如果想让windows键为mod键，可以在配置文件前加一句：set $mod Mod4</p>
